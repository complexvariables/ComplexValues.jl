ComplexValues.real_type — Method
real_type(z)

Return the base type of a real or complex float z, i.e., the type of its real part.

Examples

julia> real_type(1.0)
Float64
julia> real_type(1.0f0 + 2im)
Float32
julia> real_type(Polar(BigFloat(1), 2))
BigFloat
